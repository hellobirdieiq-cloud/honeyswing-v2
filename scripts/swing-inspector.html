Getting started

      
Produce a swing JSON with:

      
npx --yes tsx scripts/dumpPoseFull.ts <swingId>

      
Then load scripts/output/pose-full-<id8>.json via the file picker (or paste it). A bare Rtmw133Frame[] array also loads (raw 133-kp plotting only; APP-derived sections hidden).
